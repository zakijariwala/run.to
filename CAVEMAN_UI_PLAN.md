# Caveman UI Rebrand Plan

## Goal
Transform "run.to" UI from "Cyber-Dark" to "Cyber-Primal". Keep high-tech engine, use caveman words.

## UI Text Changes

### Header
- `Monolith` -> `Big Stone`
- `OSRM Online` -> `Sky Talker OK`

### Controls
- `Distance` -> `How Far`
- `Time` -> `How Long`
- `Target Distance` -> `Hunt Size`
- `Target Time` -> `Run Time`
- `Start exactly here` -> `Start at Cave`

### Results
- `Generated Route` -> `Path Found`
- `Distance` -> `Length`
- `Time` -> `Speed`
- `Pace & Splits` -> `Foot Speed`
- `Major Stops` -> `Rest Rocks`
- `Open in Google Maps` -> `Show Big Map`
- `Export GPX` -> `Carve Rock`
- `Generate Route` (Button) -> `Find Loop`

### States
- `Calculating Path...` -> `Thinking...`
- `Location Required` -> `Where Cave?`
- `Retry Access` -> `Try Again`
- `Move On` -> `Go anyway`

## Visual Tweak
- Keep dark theme.
- Ensure fonts (JetBrains Mono) still feel "carved".
