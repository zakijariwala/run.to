# run.to | Caveman Guide

## What App Do?
App make loop path for run. You tell how far, app find way back to cave.

## Big Features

### 1. Loop Maker
App use math shape to find path. Start at cave, end at cave. No get lost.

### 2. Pick Distance or Time
Tell app "5 KM" or "30 MIN". App find path match need.

### 3. Pace & Splits
App guess how fast you run. Show time for every KM. (5.5 min/km guess).

### 4. Real Road Snap
App talk to OSRM. Make sure path on real dirt, not through mountain.

### 5. Cyber Look
Dark screen. Shimmer while app think. Good for night hunt.

### 6. Export to Map
One tap send path to Google Map. Or take GPX file for Garmin rock.

### 7. Offline Work
App save in browser. No need signal in deep woods.

## How Work?
1. App draw shape around cave.
2. App ask OSRM for real road.
3. If path too long or short, app try again.
4. Scale shape until path good (<10% error).

## Future Hunt (Roadmap)
1. **Live Track:** See where you are on map while run.
2. **Run History:** App remember old paths. See past hunts.
3. **Heart Beat:** Talk to wrist rock. Log pulse.
4. **Cloud Sync:** Save history in sky cave. Access on any phone.
5. **Pro Export:** Send real run data to Strava tribe.
